HalftoneWebGL

GPU-accelerated halftone effects. No getImageData. No Three.js. Just pure shader performance at ~25KB gzipped.

1. Quick Start (Webflow)

Add the script tag to your Webflow project's custom code, then use data attributes on any div.

<!-- Add to Webflow Site Settings → Custom Code → Footer -->
<script src="https://cdn.jsdelivr.net/npm/halftone-webgl/dist/halftone-webgl.min.js"></script>
<!-- Add data-hwgl-element to any Webflow div -->
<div data-hwgl-element
     data-hwgl-source="https://your-site.com/hero.jpg"
     data-hwgl-frequency="50"
     data-hwgl-interaction="reveal"
     style="position: relative; width: 100%; height: 80vh;">
  <!-- Your Webflow content goes here (above the canvas) -->
</div>

2. JavaScript API

For full control, create an instance programmatically.

const ht = new HalftoneWebGL({
  container: '#hero',
  source: 'hero.jpg',
  frequency: 50,
  shape: 'circle',
  colorMode: 'mono',
  color: '#E85002',
  bgColor: '#050510',
  interaction: 'reveal',
  radius: 0.3,
  strength: 0.5,
});

// Update any property at runtime
ht.set('frequency', 80);
ht.set({ shape: 'diamond', contrast: 2.5 });

// Load new media
await ht.loadSource('video.mp4');
await ht.loadSource('webcam');

// Events
ht.on('ready', () => console.log('First frame rendered'));
ht.on('error', (err) => console.error(err));

// Cleanup
ht.destroy();

3. Color Modes

Switch between mono, auto (source colors), CMYK separation, and duotone.

mono + magnify
auto + warp
cmyk + ripple
duotone + vortex

4. Control from Webflow Custom Code

Access any auto-initialized instance via el.__halftoneWebGL.

<script>
// Access auto-initialized instance
const hero = document.querySelector('[data-hwgl-element]');
const ht = hero.__halftoneWebGL;

// React to Webflow interactions
hero.addEventListener('click', () => {
  ht.set('shape', ht.config.shape === 'circle' ? 'diamond' : 'circle');
});

// Listen for events
ht.on('ready', () => {
  console.log('Halftone ready!');
});
</script>
